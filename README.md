# Expense & Budget Visualizer

**CodingCamp-13april2026-RifkyAuliaArvanzaTanjung**

A mobile-friendly web app to track daily spending with visual charts and monthly summaries.

## 🚀 Live Demo
[View on GitHub Pages](#) *(update link after deployment)*

## ✅ Features (MVP)
- **Input Form** — Add transactions with Item Name, Amount, and Category (Food, Transport, Fun)
- **Transaction List** — Scrollable list with name, amount, category; delete individual items
- **Total Balance** — Live-updating total at the top of the page
- **Pie Chart** — Doughnut chart showing spending distribution by category (Chart.js)

## 🌟 Optional Challenges (3/5 implemented)
- ✅ **Custom Categories** — Add your own spending categories
- ✅ **Monthly Summary View** — Browse spending by month with category bar breakdowns
- ✅ **Sort Transactions** — Sort by newest, oldest, highest amount, lowest amount, or category
- ✅ **Spending Limit Alert** — Set a limit and get warned when you exceed it *(bonus)*
- ✅ **Dark/Light Mode Toggle** — Switch between dark and light themes *(bonus)*

## 🛠 Tech Stack
- HTML5
- CSS3 (CSS Variables, Flexbox, Grid, animations)
- Vanilla JavaScript (ES6+)
- [Chart.js](https://www.chartjs.org/) for the pie/doughnut chart
- LocalStorage API for data persistence
